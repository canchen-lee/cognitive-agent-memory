# Deep Consciousness System — Examples

Sample memory files showing the system in action.

## Contents

- `WORKING_example.md` — Working memory sample (<300 chars)
- `topics_example.md` — Episodic memory sample (topic-organized)
- `user-model_example.md` — User model sample
- `corrections_example.md` — Correction log sample

## Usage

Copy example files to `memories/` and remove the `_example` suffix:

```bash
cp examples/WORKING_example.md memories/WORKING.md
cp examples/topics_example.md memories/episodic/topics.md
cp examples/user-model_example.md memories/semantic/user-model.md
cp examples/corrections_example.md memories/semantic/corrections.md
```

Then edit each file to match your actual setup.
